# 🧠 AI Skills Architecture

Questo repository contiene un sistema modulare di skill per agenti AI, progettato per standardizzare lo sviluppo software in ambito aziendale.

## 🎯 Obiettivo

Fornire:

- processi ripetibili
- standard condivisi
- orchestrazione automatica delle competenze

## 🧩 Architettura

- `skills-index`: seleziona e orchestra le skill
- `backend/`: sviluppo API e logica server
- `frontend/`: interfacce utente
- `database/`: progettazione dati
- `external/`: skill esterne integrate

## 🔄 Workflow

1. Analisi requisiti
2. Progettazione
3. Implementazione
4. Testing
5. Report

## 🧠 Filosofia

Le skill rappresentano procedure aziendali: non si improvvisa, si segue un metodo.

## 🚀 Utilizzo

Le skill vengono caricate automaticamente dall'agente e attivate in base al contesto.
