## Scopo

Questa skill analizza la richiesta e seleziona le skill più appropriate.

## Procedura

1. Analizza il task
2. Identifica il dominio:
   - backend
   - frontend
   - database
   - sicurezza

3. Consulta `skills.yaml`
4. Seleziona le skill

5. Se necessario:
   - combina più skill
   - esegui in sequenza

## Regole

- preferisci skill interne
- mantieni coerenza
- segui sempre standard aziendali

## Output

- indica le skill usate
- esegui il task
- genera report finale
