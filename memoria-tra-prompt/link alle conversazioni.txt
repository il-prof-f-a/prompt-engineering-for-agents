🎯 La soluzione più pratica
Per il tuo caso d'uso (usare Codex/Code Interpreter con abbonamento Pro), ti consiglio:

ChatGPT-to-API (il più stabile)
bash
# Clona e installa
git clone https://github.com/pubuzhixing8/chatgpt-to-api
cd chatgpt-to-api
npm install

# Configura
cp .env.example .env
# Edita .env con le tue credenziali
# OPENAI_EMAIL=...
# OPENAI_PASSWORD=...

# Avvia
npm start
Questo crea un endpoint API OpenAI-compatibile su http://localhost:8080 che usa il tuo account Pro.

Test
bash
curl http://localhost:8080/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4-codex",
    "messages": [{"role": "user", "content": "Scrivi codice Python per Fibonacci"}]
  }'
